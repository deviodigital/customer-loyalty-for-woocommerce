== Description ==

Customer Loyalty for WooCommerce

== Installation ==

1. In your dashboard, go to `Plugins -> Add New`
2. Search for `Customer Loyalty for WooCommerce` and install this plugin
3. Pat yourself on the back for a job well done :)
